Download this project Download amber 0.12.3 and save it somewhere Follow instructions from amber-lang.net Wiki (getting depencies and so on...) Go to "this project"/client folder Create symbolic link to amber directory, name it, for example, "vendor" (this name is on .gitignore file) Run file server using 'vendor/bin/amber serve' command (like in instructions on amber wiki) Run server image with Pharo. Open your browser and go to 'localhost:4000/index.html' If all will be alright, browser said you that connection was made successfuly.
